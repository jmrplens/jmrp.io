## Using Custom Components

### Callout Component
Use callouts to highlight important information:

<Callout type="info">
  This is an info callout. Available types: info, warning, danger, success
</Callout>

### Code Tabs
Show multiple code examples side by side:

<CodeTabs>
  <CodeTabItem label="JavaScript">
    ```javascript
    console.log('Hello World');
    ```
  </CodeTabItem>
  <CodeTabItem label="Python">
    ```python
    print("Hello World")
    ```
  </CodeTabItem>
</CodeTabs>

### YouTube Embed
<YouTube id="VIDEO_ID_HERE" title="Video Title" />

### References Section
Add at the end of your post to display all references:

<References references={frontmatter.references} />
